📈 Trade-off Analysis

Latency vs Accuracy: The RAG retrieval step adds ~20–80ms but ensures answers are grounded in your documents — preventing hallucinations. Without RAG, the LLM answers from training data only.

Cost vs Performance: Sarvam-M is currently free, making this system extremely affordable. Primary costs are STT (₹30/hr audio) and TTS (₹30/10K chars). Local embeddings eliminate all embedding API costs.

Open-Source Alternative: Replace Sarvam-M with Whisper (STT) + Llama 3 (LLM) + MeloTTS (TTS) for near-zero cost, at the expense of ~2–5× higher latency and more infrastructure work.
